# 中文已发表文章引注规则

> 对应《法学引注手册》第二版第25-26条（文献信息依据）、第29条（翻译文章）、第42-49条（文章引注）

---

## 总则：文献信息依据（第25-26条）

- [ ] 图书信息以版权页为准，论文信息以首页为准
- [ ] 版权页信息有误时，可参考其他权威信息并予以说明

---

## 一、学术期刊文章（第43条）

### 基本格式

**格式规范**：作者：《文章名》，载《期刊名》年份第X期，第X页。

- [ ] 作者姓名后紧跟冒号（全角冒号 `：`），无空格
- [ ] 文章标题用书名号《》括注
- [ ] 书名号后**无需**标点，直接以"载"字连接期刊名称
- [ ] "载"字后紧跟期刊名称，**无空格**
- [ ] 期刊名称用书名号《》括注
- [ ] 期刊所属"学报""法学版""社会科学版""哲学社会科学版"等属于期刊名称的组成部分，**写在书名号内**
- [ ] 出版信息格式为"年份第X期"，**期号前不加"总第X期"**（总第期号不应取代期号）
- [ ] 页码格式为"第X页"或"第X-X页"，**页码前有"第"字**
- [ ] 页数起止用短横线 `-` 连接，不得用一字线 `—` 或半字线 `–`
- [ ] 格式结尾为句号（全角句号 `。`）

### 期刊名称示例

| 正确写法 | 错误写法 |
|---------|---------|
| 《清华大学学报（哲学社会科学版）》 | 清华大学学报（哲学社会科学版） |
| 《中国法学》 | 中国法学 |
| 《法学研究》 | 法学研究 |
| 《华东政法大学学报》 | 华东政法大学学报 |
| 《北大法律评论》 | 北大法律评论 |

### 期刊更名

- [ ] 引用时写**发表当时**的期刊名称
- [ ] 如《法学研究》曾用名《政法研究》，1994年更名，则1994年前的文章应写《政法研究》

### 正误示例

> **错误示例**：张明楷：《刑法学的思路》，中国法学2020年第3期，45页。
>
> 错误原因：缺"载"字，期刊名缺书名号，页码前缺"第"字。

> **正确示例**：张明楷：《刑法学的思路》，载《中国法学》2020年第3期，第45页。

> **错误示例**：王利明：《物权法的基本原则》，载《法学家》2020总第175期，第12页。
>
> 错误原因：不应写"总第X期"，应直接写"第X期"（除非"总"是期刊期号体系的正式组成部分且手册允许）。

> **正确示例**：王利明：《物权法的基本原则》，载《法学家》2020年第2期，第12页。

> **错误示例**：陈兴良：《正当防卫论》，载《法学研究》2020年第三期，第23页。
>
> 错误原因：期号应使用阿拉伯数字，不应使用汉字数字。

> **正确示例**：陈兴良：《正当防卫论》，载《法学研究》2020年第3期，第23页。

> **错误示例**：崔建远：《合同法》，载《清华法学》2019年第5期，第23—34页。
>
> 错误原因：页码起止应使用短横线 `-`，不应使用一字线 `—`。

> **正确示例**：崔建远：《合同法》，载《清华法学》2019年第5期，第23-34页。

> **错误示例**：作者在1995年引用1979年《法学研究》上的文章，写为载《政法研究》1979年第X期。
>
> 正确做法：如《法学研究》与《政法研究》是同一本期刊更名而来，应写发表当时的名称。但若确属不同期刊，则按事实标注。

## 二、集刊中的文章（第44条）

**格式规范**：作者：《文章名》，载主编：《集刊名》第X卷/第X辑，出版社年份，第X页。

- [ ] 作者名后接冒号和文章名
- [ ] 文章名用书名号
- [ ] 集刊主编姓名后接冒号，再写集刊名
- [ ] 集刊名用书名号
- [ ] 标明卷号或辑号：格式为"第X卷"或"第X辑"
- [ ] 出版社信息：出版社名称加出版年份
- [ ] 页码：第X页
- [ ] 如果版权页未标明主编，则**不写主编**信息，直接写：作者：《文章名》，载《集刊名》第X卷，出版社年份，第X页。

### 正误示例

> **错误示例**：张千帆：《司法独立》，载《北大法律评论》第5卷，第12页。
>
> 错误原因：集刊需要标注主编信息（除非版权页未标明）。

> **正确示例**：张千帆：《司法独立》，载《北大法律评论》编辑委员会：《北大法律评论》第5卷，北京大学出版社2003年版，第12页。

> **正确示例**（无主编）：张千帆：《司法独立》，载《北大法律评论》第5卷，北京大学出版社2003年版，第12页。

## 三、报纸文章（第45条）

**格式规范**：作者：《文章名》，载《报纸名》年月日，第X版。

- [ ] 作者姓名后接冒号
- [ ] 文章标题用书名号
- [ ] 报纸名称用书名号
- [ ] 日期格式：年月日连续书写，不添加空格或标点，如"2023年5月10日"
- [ ] 版号格式："第X版"，**不加"0"**，不写"第02版"，应写"第2版"
- [ ] 报纸标题包含两部分且无主从关系时，两部分之间用一个字符空格隔开（如《北京日报 京华视点》）

### 正误示例

> **错误示例**：姜明安：《论行政法治》，人民日报2023-05-10，第02版。
>
> 错误原因：缺"载"字，报纸名缺书名号，日期连字符格式不当，版号"02"不应有前导零。

> **正确示例**：姜明安：《论行政法治》，载《人民日报》2023年5月10日，第2版。

> **错误示例**：应松年：《依法行政的意义》，载《人民法院报》2022年12月5日，06版。
>
> 错误原因：版号"06"不应有前导零。

> **正确示例**：应松年：《依法行政的意义》，载《人民法院报》2022年12月5日，第6版。

## 四、文集中的文章（第42条）

**格式规范**：作者：《文章名》，载编者/主编：《文集名》，出版社年份，第X页。

- [ ] 作者姓名后接冒号和文章名
- [ ] 文章名用书名号
- [ ] 编者姓名后接冒号
- [ ] 文集名用书名号
- [ ] 出版社名称加出版年份
- [ ] 页码：第X页

### 翻译文章在文集中

**格式规范**：作者：《文章名》，译者译，载编者/编译者：《文集名》，出版社年份，第X页。

- [ ] 在作者和文章名之后，注明译者信息："×××译"
- [ ] 译者后接逗号（全角）
- [ ] 然后写"载"字
- [ ] 编者/编译者信息："×××编"或"×××编译"
- [ ] 文集名用书名号
- [ ] 出版社年份，第X页

### 正误示例

> **错误示例**：沈岿：《行政行为》，载罗豪才主编《行政法论丛》法律出版社2020年第22页。
>
> 错误原因：主编与文集名之间缺冒号，出版社前缺逗号，页码前应有逗号。

> **正确示例**：沈岿：《行政行为》，载罗豪才主编：《行政法论丛》第3卷，法律出版社2020年版，第22页。

> **错误示例**：[德]卡尔·拉伦茨：《法学方法论》，载《民法学说与判例研究》。
>
> 错误原因：翻译文章未标注译者；文集信息不完整（缺编者、出版社、年份、页码）。

> **正确示例**：[德]卡尔·拉伦茨：《法学方法论》，陈爱娥译，载王泽鉴编：《民法学说与判例研究》第1册，中国政法大学出版社2005年版，第33页。

## 五、翻译文章（第29条关于文章部分）

**格式规范**：[国籍]作者：《文章名》，译者译，载《期刊名》年份第X期，第X页。

- [ ] 作者国籍用六角括号〔〕括注（如〔日〕、〔德〕、〔美〕），**非方括号[]**或圆括号（）
- [ ] 作者姓名
- [ ] 冒号和文章名
- [ ] 文章名用书名号
- [ ] 译者信息：姓名后加"译"
- [ ] "载"字
- [ ] 期刊名用书名号
- [ ] 年份第X期
- [ ] 页码：第X页

### 正误示例

> **错误示例**：[日]我妻荣：《债权法理论》，载《民法研究》2020年。
>
> 错误原因：国籍应使用六角括号〔〕，缺译者，缺期号，缺页码。

> **正确示例**：〔日〕我妻荣：《债权法理论》，王轶译，载《民法研究》2020年第1期，第45页。

---

## 六、再次引用原则（结合第13条）

- [ ] 同一篇文章中再次引用同一文献时，采用"同前注""同上注"等略写
- [ ] 具体规则参见 `rules-repeated-citation.md`

---

## 综合检查清单

引用中文文章时，逐项核查以下要素：

- [ ] 作者姓名与文章标题之间是否有全角冒号
- [ ] 文章标题是否使用书名号《》
- [ ] "载"字是否在恰当位置（文章名与期刊/文集/报纸名之间）
- [ ] 期刊/文集/报纸名称是否使用书名号
- [ ] 出版信息格式是否正确（年份第X期 / 出版社年份）
- [ ] 页码格式是否正确（第X页或第X-X页）
- [ ] 标点符号是否为全角中文标点
- [ ] 页码起止是否使用短横线 `-`
- [ ] 翻译文章是否有国籍标记（六角括号〔〕）和译者信息
- [ ] 集刊文章是否标注了主编信息（或确认版权页未标明主编）
- [ ] 报纸文章的版号是否无前导零
- [ ] 期刊名称中的"学报""版"等是否在书名号内
