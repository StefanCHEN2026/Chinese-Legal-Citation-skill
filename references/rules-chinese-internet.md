# 中文网络电子文献引注规则

> 对应《法学引注手册》第二版第50-58条

---

## 一、引用原则（第50条）

- [ ] **优先引用纸质版**：同一文献既有纸质版又有网络版时，优先引用纸质版
- [ ] **引首发网站**：同一文献被多个网站转载时，引用首发网站
- [ ] **不引已消失网页**：已确认无法访问的网页不引用
- [ ] **动态内容不引**：内容可能动态变化的页面（如实时行情、动态榜单）原则上不引用

### 正误示例

> **错误示例**：引用一篇已在《中国法学》发表的论文时，仅引用知网链接。
>
> 正确做法：应引用纸质版《中国法学》上的原文，网络版可作为辅助信息。

> **错误示例**：引用一个已知关闭的政府网站上的文件。
>
> 正确做法：该网页已消失，应寻找其他可靠来源或不予引用。

---

## 二、基本格式（第51条）

**格式规范**：作者：《文章名》，载网站名 上传日期，链接。

- [ ] 作者：《文章名》的格式与纸质文章相同（作者名后全角冒号，文章名用书名号）
- [ ] 文章名后无标点，直接接"载"字
- [ ] 网站名后**不加逗号**，直接空格后接日期（网站名与日期之间用空格分隔）
- [ ] 日期与链接之间**加逗号**（全角逗号）
- [ ] 链接以 `http://` 或 `https://` 开头
- [ ] 结尾句号
- [ ] 如果链接中已标日期，可**不重复标注**上传日期

### 正误示例

> **错误示例**：王利明：《民法典的意义》，载中国法院网，2021年3月15日，https://example.com。
>
> 错误原因：网站名"中国法院网"后不应有逗号。

> **正确示例**：王利明：《民法典的意义》，载中国法院网 2021年3月15日，https://example.com。

> **正确示例**（链接中已有日期）：王利明：《民法典的意义》，载中国法院网，https://example.com/2021/0315/article.html。
>
> 说明：链接URL中已包含日期信息，可省略单独标注上传日期。

---

## 三、作者（第52条）

- [ ] 作者使用**网名**时直接写网名
- [ ] 必要时在网名后括注真实姓名，如"秋风（姚中秋）"
- [ ] **无作者信息**时，直接以文章名开头，省略作者部分
- [ ] 作者信息不明的，不写"佚名"或"无名"

### 正误示例

> **错误示例**：佚名：《论司法改革》，载某网站 ...
>
> 错误原因：不应使用"佚名"替代作者，无作者信息时直接以文章名开头。

> **正确示例**：秋风（姚中秋）：《重新理解法治》，载思想库网 2010年9月1日，http://example.com。

---

## 四、上传日期（第53条）

- [ ] 上传日期标注在**网站名之后、链接之前**
- [ ] 网站名与上传日期之间为**空格**（无逗号）
- [ ] 日期格式：×年×月×日（如"2023年5月10日"）
- [ ] 只有年份的写年份，年份月份都有的写年月
- [ ] 链接中已包含日期的，可省略上传日期

### 正误示例

> **错误示例**：载某网站，2023年5月10日，http://example.com。
>
> 错误原因：网站名后多写了逗号，应删除逗号。

> **正确示例**：载某网站 2023年5月10日，http://example.com。

---

## 五、访问日期（第54条）

- [ ] **仅在以下情况**标注访问日期：
  - [ ] 网页**无上传日期**
  - [ ] 涉及**内容可能更新的动态页面**
- [ ] 格式："×年×月×日访问"
- [ ] **不写**"最后访问"、"accessed on"等用词
- [ ] 访问日期放在链接之后

### 正误示例

> **错误示例**：载某网站，http://example.com，2023年5月10日最后访问。
>
> 错误原因：不应写"最后访问"，应写"×年×月×日访问"；且访问日期不应在有上传日期时使用。

> **正确示例**：载某网站，http://example.com（2023年5月10日访问）。
>
> 说明：网页无上传日期时使用访问日期。

---

## 六、链接（第55条）

- [ ] 链接格式：使用 `http://` 或 `https://` 完整格式
- [ ] 链接过长的可以不标注（由引用者自行判断）
- [ ] 链接不应包含跟踪参数（如 `utm_source` 等）

---

## 七、博客文章（第56条之一）

**格式规范**：作者：《文章名》，载博客名 上传日期，链接。

- [ ] 作者：《文章名》与一般文章格式相同
- [ ] 写"载博客名"（博客名称不加书名号）
- [ ] 上传日期
- [ ] 逗号后加链接

### 正误示例

> **错误示例**：贺卫方：《司法改革》，载《法律博客》2010年5月1日，http://example.com。
>
> 错误原因：博客名不应加书名号。

> **正确示例**：贺卫方：《司法改革》，载法律博客 2010年5月1日，http://example.com。

---

## 八、微博（第56条之二）

**格式规范**：微博账号，上传日期，链接。

- [ ] 以微博账号（用户名）开头，**不加冒号**
- [ ] 微博名称用**微博账号本身**，不写真实姓名（除非必要）
- [ ] 不写"载"字
- [ ] 逗号后写上传播日期
- [ ] 逗号后写链接
- [ ] 一般**不写文章名**，如果确需写明微博内容主题，可加简要说明

### 正误示例

> **错误示例**：张三：《论司法改革》，载微博 2023年5月10日，https://weibo.com/xxx。
>
> 错误原因：微博引注不以文章名开始，不加"载"字。

> **正确示例**：最高人民法院，2023年5月10日，https://weibo.com/xxx。

---

## 九、微信公众号文章（第56条之三）

**格式规范**：作者：《文章名》，载微信公众号"名称" 上传日期。

- [ ] 作者：《文章名》格式与一般文章相同
- [ ] 写"载微信公众号"（不加书名号）
- [ ] 公众号名称用**引号""**（全角/直角引号均可，建议全角双引号）
- [ ] 上传日期
- [ ] **不要求**标注链接（因为微信文章链接通常长期有效且可搜索到）
- [ ] 原文无上传日期但有发布日期时，标发布日期

### 正误示例

> **错误示例**：载《微信公众号》中国普法 2023年5月10日。
>
> 错误原因：不应使用书名号标注公众号名称。

> **正确示例**：张明楷：《刑法修正案解读》，载微信公众号"中国普法" 2023年5月10日。

> **正确示例**（无作者）：《最高法发布司法解释》，载微信公众号"最高人民法院" 2023年5月10日。

---

## 十、App文章（第56条之四）

**格式规范**：文章名，载"App名" 上传日期。

- [ ] 以文章名开头，**不写作者**（除非确有必要）
- [ ] 写"载App名"
- [ ] App名称用**引号**
- [ ] 上传日期

### 正误示例

> **错误示例**：张三：《今日说法》，载App"央视新闻" 2023年5月10日。
>
> 错误原因：App文章一般不写作者，直接以文章名开头。

> **正确示例**：《今日说法热点解读》，载"央视新闻"App 2023年5月10日。

---

## 十一、广播电视节目（第57条）

**格式规范**：主持人：《节目名》，电视台"栏目名"播出日期播出，链接。

- [ ] 以主持人姓名开头，后接冒号
- [ ] 节目名用书名号
- [ ] 电视台名称
- [ ] 栏目名用引号
- [ ] "播出日期播出"（重复"播出"二字）
- [ ] 如有网络回看链接，加逗号后标注链接

### 正误示例

> **错误示例**：央视《今日说法》，2023年5月10日播出。
>
> 错误原因：缺主持人（如有明确主持人），缺电视台名称后栏目名引号。

> **正确示例**：李晓东：《普法进行时》，中央电视台"社会与法" 2023年5月10日播出，https://example.com。

---

## 十二、音像制品（第58条）

**格式规范**：名称、制作单位、出版时间。

- [ ] 标注音像制品的**名称**
- [ ] 标注**制作单位**
- [ ] 标注**出版时间**
- [ ] 要素之间用逗号（全角）分隔
- [ ] 分类说明：如有必要，可在名称后括注介质类型（如DVD、CD-ROM）

### 正误示例

> **错误示例**：《民法典讲座》DVD。
>
> 错误原因：缺制作单位、缺出版时间。

> **正确示例**：《民法典理解与适用》（DVD），法律出版社音像部，2020年。

---

## 综合检查清单

引用网络电子文献时，逐项核查以下要素：

- [ ] 已确定纸质版不存在，或者有合理理由引用网络版
- [ ] 如果有多个网站转载，引用了首发网站
- [ ] 网页当前可正常访问
- [ ] 作者信息处理正确（网名/实名/省略）
- [ ] 网站名后无逗号，直接空格接日期
- [ ] 日期格式为"×年×月×日"
- [ ] 链接格式为 `http://` 或 `https://` 开头
- [ ] 日期与链接之间用逗号分隔
- [ ] 仅必要时标注访问日期（无上传日期或动态页面）
- [ ] 访问日期格式为"×年×月×日访问"
- [ ] 非必要情况下未使用"最后访问"
- [ ] 微博引注不以文章名开头
- [ ] 微信公众号名称使用引号，不使用书名号
- [ ] App文章一般不标注作者
- [ ] 音像制品标注了名称、制作单位、出版时间
