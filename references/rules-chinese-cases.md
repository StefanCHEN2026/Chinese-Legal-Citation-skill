# 中文司法案例引注规则

> 对应《法学引注手册》第二版第87-94条
>
> 优先级：判定为司法案例类型后，按以下编号逐条检查。

---

## 第87条 一般规范

### 优先援引原则

- [ ] 引用司法案例时，优先援引正式的裁判文书
- [ ] 裁判文书包括判决书、裁定书、决定书、调解书等
- [ ] 应使用官方公布的文书版本

### 基本格式要素

- [ ] 司法案例引注基本格式包含以下要素：
  - 当事人 + 诉 + 当事人 + 案由 + 案
  - 法院全称 + 案号 + 文书类型

| 错误示例 | 正确示例 |
|---------|---------|
| 张三诉李四案（缺少法院、案号等关键要素） | 张三诉李四民间借贷纠纷案，北京市海淀区人民法院（2018）京0108民初142号民事判决书 |
| 王某诉某公司案（当事人信息过于模糊，无案号） | 王某某诉北京某科技有限公司合同纠纷案，北京市朝阳区人民法院（2019）京0105民初12345号民事判决书 |

## 第88条 案例名称

### 民事、行政案件名称格式

- [ ] 格式：`原告 + 诉 + 被告 + 案由 + 案`
- [ ] 原告和被告之间用"诉"连接（不用"v."或"vs."）

| 错误示例 | 正确示例 |
|---------|---------|
| 张三 v. 李四 合同纠纷案（使用英文"v."） | 张三诉李四合同纠纷案 |
| 张三 vs 李四借贷案（使用"vs."） | 张三诉李四民间借贷纠纷案 |

### 刑事案件名称格式

- [ ] 格式：`被告人 + 被指控罪名 + 案`
- [ ] 开头用"被告人"而非犯罪嫌疑人

| 错误示例 | 正确示例 |
|---------|---------|
| 张三盗窃案（缺少"被告人"字样） | 被告人张三盗窃案 |
| 犯罪嫌疑人李四故意杀人案（用词不规范） | 被告人李四故意杀人案 |

### 当事人姓名处理

- [ ] 当事人使用真实姓名
- [ ] 涉及未成年人、隐私案件等依法需要保密的，使用化名或隐名处理
- [ ] 隐名格式：`张某某`、`王某某` 或 `张某`、`王某`

| 错误示例 | 正确示例 |
|---------|---------|
| 张三诉李四离婚案（离婚案件当事人应做隐名处理） | 张某诉李某离婚纠纷案 |
| 被告人王小明强奸案（涉及隐私的刑事案件应化名） | 被告人王小某强奸案 |

### 多当事人处理

- [ ] 多个当事人的，只列第一个当事人，后加"等"字
- [ ] 同一案件中有多个原告或多个被告的，不逐一列举

| 错误示例 | 正确示例 |
|---------|---------|
| 张三、李四、王五、赵六诉某公司合同纠纷案（当事人过多，累赘） | 张三等诉某公司合同纠纷案 |
| 被告人赵某、钱某、孙某、李某、周某、吴某、郑某盗窃案（多被告人全列） | 被告人赵某等盗窃案 |

### 案由处理

- [ ] 案由可从简，直接写明主要的法律关系即可
- [ ] 案由过长时可以省略或简化

| 错误示例 | 正确示例 |
|---------|---------|
| 张三诉李四买卖合同纠纷案（案由简化适当） | 张三诉李四买卖合同案 |
| 张三诉李四股权转让纠纷案（完整案由亦可用） | 张三诉李四股权转让纠纷案 |

### 案例简称

- [ ] 众所周知或具有重大影响的案例可以使用简称
- [ ] 简称格式：在案例后加"案"字或"案（简称）"
- [ ] 简称应在首次引用全称后括注

| 错误示例 | 正确示例 |
|---------|---------|
| 刘燕文案件（简称不规范） | "刘燕文案"（北京大学诉刘燕文学位纠纷案） |
| 于欢案首次引用直接使用简称 | 山东省聊城市中级人民法院（2016）鲁15刑初33号刑事判决书（"于欢案"） |

## 第89条 案号

### 案号基本格式

- [ ] 格式：`法院全称 + （年份） + 案号 + 文书类型`
- [ ] 法院名称使用全称，不得省略或使用简称
- [ ] 年份与前后的内容之间有一个空格

| 错误示例 | 正确示例 |
|---------|---------|
| 北京海淀法院(2018)142号（法院使用简称，缺文书类型） | 北京市海淀区人民法院（2018）京0108民初142号民事判决书 |
| 最高院(2016)最高法民再1号（法院简称"最高院"不规范） | 最高人民法院（2016）最高法民再1号民事判决书 |

### 年份括号格式

- [ ] 案号中的年份使用**圆括号** `（）`
- [ ] 不得使用方括号 `[]` 或六角括号 `〔〕`

| 错误示例 | 正确示例 |
|---------|---------|
| 北京市海淀区人民法院〔2018〕京0108民初142号（使用六角括号） | 北京市海淀区人民法院（2018）京0108民初142号 |
| 北京市海淀区人民法院[2018]京0108民初142号（使用方括号） | 北京市海淀区人民法院（2018）京0108民初142号 |

### 2016年起新案号规则

- [ ] 2016年起实行的新案号格式不带"字""第"字样
- [ ] 旧案号（2016年前）保留"字""第"等要素
- [ ] 新案号格式：`（年份）代码+类型+序号`

| 错误示例 | 正确示例 |
|---------|---------|
| （2018）京0108民初字第142号（2016年后的案号仍带"字""第"） | （2018）京0108民初142号（2016年后新案号格式） |
| （2015）海民初字第12345号（2016年前案号可保留"字""第"） | （2015）海民初字第12345号（2016年前保留原格式） |

### 代码前导零处理

- [ ] 法院代码开头的"0"可以省略
- [ ] 例如"京0108"可以写为"京108"，但应保持全文一致

| 错误示例 | 正确示例 |
|---------|---------|
| （2018）京0108民初142号（保留前导零，不算错误，但可省略） | （2018）京108民初142号（省略前导零） |
| 同一篇文章中前导零有时省略有时不省略（不一致） | 全文统一，如统一用"京0108"或统一用"京108" |

### 文书类型

- [ ] 案号后应注明文书类型，如"民事判决书""民事裁定书""刑事判决书"等
- [ ] 文书类型不可省略

| 错误示例 | 正确示例 |
|---------|---------|
| 最高人民法院（2016）最高法民再1号（缺少文书类型） | 最高人民法院（2016）最高法民再1号民事判决书 |
| 北京市海淀区人民法院（2018）京0108民初142号判决（"判决"不规范，应写全称） | 北京市海淀区人民法院（2018）京0108民初142号民事判决书 |

## 第90条 案例来源

### 指导性案例

- [ ] 格式：`发布机关 + 指导案例第X号 + （年份）`
- [ ] 发布机关通常为"最高人民法院"或"最高人民检察院"
- [ ] 案例编号用阿拉伯数字，年份用圆括号

| 错误示例 | 正确示例 |
|---------|---------|
| 最高法指导案例24号（发布机关简称不规范，缺少年份） | 最高人民法院指导案例24号（2014年） |
| 最高人民检察院指导性案例第X号（"第"字多余） | 最高人民检察院指导案例第X号 |
| 指导案例10号（缺少发布机关） | 最高人民法院指导案例10号（2012年） |

### 公报案例

- [ ] 格式：`载《最高人民法院公报》年份第X期`
- [ ] "公报"加书名号
- [ ] 注明刊载的期号

| 错误示例 | 正确示例 |
|---------|---------|
 载最高人民法院公报2018年第3期（"公报"未加书名号） | 载《最高人民法院公报》2018年第3期 |
| 载最高法公报2018年第3期（法院简称不规范） | 载《最高人民法院公报》2018年第3期 |

### 人民法院案例库

- [ ] 格式：标注入库编号
- [ ] 注明"人民法院案例库"字样

| 错误示例 | 正确示例 |
|---------|---------|
| 张三诉李四案（仅案号，未注明入库来源） | 张三诉李四民间借贷纠纷案，北京市海淀区人民法院（2018）京0108民初142号民事判决书（人民法院案例库编号：2023-01-2-123-001） |
| 案例库案例仅写案号（缺少入库编号） | （人民法院案例库编号：2023-01-2-123-001） |

### 媒体报道

- [ ] 引用媒体报道的案例，应标明媒体名称、报道日期、标题
- [ ] 可附链接

| 错误示例 | 正确示例 |
|---------|---------|
| 据报道，张三诉李四案……（媒体信息过于模糊） | "张三诉李四案一审宣判"，载《人民法院报》2020年5月20日，http://××× |
| 某媒体报道了该案（未注明具体媒体和日期） | 李某某诉某公司案，载《南方周末》2019年8月15日，第A3版 |

### 案例选编

- [ ] 引用案例选编中的案例，应标明选编出版信息
- [ ] 格式：案例名称 + 载《选编名称》出版社年份版第X页

| 错误示例 | 正确示例 |
|---------|---------|
| 该案载于《案例选编》（出版信息不完整） | 张三诉李四案，载《中国审判案例要览（2018年民事审判案例卷）》，中国人民大学出版社2019年版，第100-105页 |
| 《刑事审判参考》案例（缺少期号和页码） | 王某某抢劫案，载《刑事审判参考》第120集，法律出版社2019年版，第50-60页 |

## 第91条 裁判时间

### 一般规则

- [ ] 裁判时间一般不需要在引注中出现
- [ ] 案号中的年份已表明裁判作出的年份

| 错误示例 | 正确示例 |
|---------|---------|
| 北京市海淀区人民法院2018年5月10日（2018）京0108民初142号民事判决书（裁判日期冗余） | 北京市海淀区人民法院（2018）京0108民初142号民事判决书 |

### 必要时标注

- [ ] 确有必要时（如区分不同审级的裁判、同一年份同案号等），可以标注裁判时间
- [ ] 时间应以裁判文书的落款日期为准
- [ ] 格式：`YYYY年M月D日`

| 错误示例 | 正确示例 |
|---------|---------|
| 最高人民法院（2016）最高法民再1号民事判决书（2016.3.10）（日期格式不规范） | 最高人民法院（2016）最高法民再1号民事判决书（2016年3月10日） |
| 北京市海淀区人民法院（2018）京0108民初142号民事判决书（2018-05-10）（日期格式不规范） | 北京市海淀区人民法院（2018）京0108民初142号民事判决书（2018年5月10日） |

## 第92-94条 统计数据

### 第92条 统计数据标明出处

#### 基本原则

- [ ] 除周知事实外，所有统计数据应标明出处
- [ ] 周知事实指广泛知晓、无需证明的数据（如中国有14亿人口、一年有365天等）
- [ ] 非周知数据的出处应当具体、可查证

| 错误示例 | 正确示例 |
|---------|---------|
| 据统计，2023年全国法院受理案件数量大幅上升（未标明来源） | 据最高人民法院工作报告，2023年全国各级法院受理案件4557.43万件（参见周强《最高人民法院工作报告》（2024年3月8日在第十四届全国人民代表大会第二次会议上）） |
| 某地区刑事案件发案率下降20%（来源不详） | 根据国家统计局《中国统计年鉴2023》，2022年全国刑事案件立案率较2018年下降约15%。 |

#### 来源格式

- [ ] 统计数据来源的引注格式遵循其所出文献类型的规则
- [ ] 来源应为书、文章、报告、官方统计数据等

| 错误示例 | 正确示例 |
|---------|---------|
| 数据来源于网络（过于模糊，无法查证） | 数据来源于国家统计局《中国统计年鉴2023》，中国统计出版社2023年版，第250页。 |
| 最高人民法院公布的数据显示（未注明具体报告名称和发布时间） | 参见最高人民法院《全国法院司法统计公报》（2023年），载最高人民法院网站，https://××× |

### 第93条 数据可靠性与合理性

#### 来源可靠性

- [ ] 统计数据应来自可靠、权威的渠道
- [ ] 优先使用官方统计、权威机构发布的数据
- [ ] 使用第三方数据时，评估其可信度

| 错误示例 | 正确示例 |
|---------|---------|
| 据某网络论坛统计，离婚率已超过50%（来源不可靠） | 据民政部《2023年民政事业发展统计公报》，2023年全国结婚登记768.2万对，离婚登记259.3万对。 |
| 据微博用户发布的数据（来源不权威） | 据国家统计局官方网站数据（https://×××） |

#### 数据合理性

- [ ] 对数据的合理性进行判断，数据应符合常识和逻辑
- [ ] 如数据明显异常，应进一步核实

| 错误示例 | 正确示例 |
|---------|---------|
| "全国法院案件量增长300%"（数据明显异常，未核实即引用） | 引用前核实数据，确保引用准确无误。如"全国法院案件量增长3%" |

### 第94条 图表呈现

#### 图表基本要求

- [ ] 使用图表呈现统计数据时，图表应有标题和编号
- [ ] 图表标题应简明扼要，概括图表内容
- [ ] 图表编号应与正文引用对应

| 错误示例 | 正确示例 |
|---------|---------|
| （文中插入一张图表，无标题无编号） | 如图1所示，2019-2023年全国法院收案数量呈逐年上升趋势。|
| （正文引用"如下图所示"，但图无编号） | （见图2-1：2019-2023年全国法院收案数量统计） |

#### 图表标题格式

- [ ] 图表标题格式：`图X` 或 `表X` + 标题文字
- [ ] 图标题通常放在图的下方，表标题通常放在表的上方
- [ ] 图表编号可连续编号（图1、图2）或按章编号（图2-1、表3-1）

| 错误示例 | 正确示例 |
|---------|---------|
| （图表标题为"统计图"，过于模糊） | 图1 2019-2023年全国法院民事一审收案数量 |
| （表头"各类案件数量"，无编号） | 表2-1 2023年全国法院各类案件收案情况统计 |

## 综合检查清单

- [ ] 案例名称格式是否正确（民事：原告诉被告案由案；刑事：被告人指控罪名案）？
- [ ] 当事人姓名是否真实，隐私案件是否隐名处理？
- [ ] 多当事人是否只写第一个加"等"？
- [ ] 案由是否简洁适当？
- [ ] 法院名称是否使用全称？
- [ ] 案号年份是否使用圆括号 `（）`？
- [ ] 2016年后案号是否去掉了"字""第"？
- [ ] 案号后是否注明了文书类型？
- [ ] 案例来源是否明确标注（指导性案例、公报案例、案例库编号等）？
- [ ] 裁判时间仅在必要时标注，格式是否正确？
- [ ] 统计数据是否标明出处？
- [ ] 数据来源是否可靠、数据是否合理？
- [ ] 图表是否有标题和编号？
