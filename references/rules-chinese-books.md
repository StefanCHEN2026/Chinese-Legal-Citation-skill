# 中文图书引注规则（第25-49条、第64条）

> 以下为检查清单格式，每条规则以 `- [ ]` 开头，附正确与错误示例。

---

## A. 文献作者（第27-36条）

- [ ] **第27条：原创作品省略"著"，翻译/编辑/整理作品标明著作方式**
  - 正确（原创）：王泽鉴：《民法总则》，北京大学出版社2009年版，第15页。
  - 正确（翻译）：[德]卡尔·拉伦茨：《法学方法论》，陈爱娥译，商务印书馆2003年版，第50页。
  - 正确（编辑）：梁慧星主编：《民商法论丛》第1卷，法律出版社1994年版，第1页。
  - 错误（原创加"著"）：王泽鉴著：《民法总则》，北京大学出版社2009年版，第15页。
  - 错误（翻译未标著作方式）：[德]卡尔·拉伦茨：《法学方法论》，陈爱娥译，商务印书馆2003年版，第50页。（漏标"译"）

- [ ] **第28条：编辑作品加"主编""编""编著""编译"**
  - 正确：梁慧星主编：《中国民法典草案建议稿》，法律出版社2003年版。
  - 正确：法学教材编辑部编：《中国民法教程》，人民法院出版社1995年版。
  - 错误：梁慧星：《中国民法典草案建议稿》，法律出版社2003年版。（未标注"主编"，混淆个人作品与主编作品）

- [ ] **第29条：翻译作品译者位于书名后、出版信息前，格式：[国籍]作者：《书名》，译者译，出版社版次**
  - 正确：[德]卡尔·拉伦茨：《法学方法论》，陈爱娥译，商务印书馆2003年版。
  - 正确：[日]我妻荣：《债权在近代法中的优越地位》，王书江等译，中国大百科全书出版社1999年版。
  - 错误：[德]卡尔·拉伦茨：《法学方法论》，商务印书馆2003年版，陈爱娥译。（译者位置错误，应位于书名后、出版社前）
  - 错误：[德]卡尔·拉伦茨：《法学方法论》（陈爱娥译），商务印书馆2003年版。（不应将译者信息括注）

- [ ] **第30条：口述访谈作品整理者**
  - 正确：陈光中口述：《中国刑事诉讼法制四十年》，陈夏红整理，中国政法大学出版社2019年版。
  - 正确：江平口述：《沉浮与枯荣：八十自述》，陈夏红整理，法律出版社2010年版。
  - 错误：陈光中：《中国刑事诉讼法制四十年》，中国政法大学出版社2019年版。（未标"口述"和"整理"，混淆著作权关系）

- [ ] **第31条：勘校作品点校者**
  - 正确：（清）沈家本：《历代刑法考》，邓经元、骈宇骞点校，中华书局1985年版。
  - 错误：（清）沈家本：《历代刑法考》，中华书局1985年版。（未标注点校者）

- [ ] **第32条：作者一般写法（直写姓名、全名、团体作者用编写组名、笔名从笔名）**
  - 正确（直写姓名）：王泽鉴：《民法总则》，北京大学出版社2009年版。
  - 正确（团体作者）：最高人民法院民法典贯彻实施工作领导小组主编：《中华人民共和国民法典总则编理解与适用》，人民法院出版社2020年版。
  - 正确（编写组）：《中国民法典释评》编写组：《中国民法典释评》，中国人民大学出版社2020年版。
  - 正确（笔名）：鲁迅：《中国小说史略》，北新书局1925年版。
  - 错误（团体作者简化不当）：最高院编：《民法典理解与适用》，人民法院出版社2020年版。（应用全称）

- [ ] **第33条：外籍作者用方括号[国籍]，多个同国籍只标第一个，不同国籍分别标注**
  - 正确（单国籍）：[德]卡尔·拉伦茨：《法学方法论》，陈爱娥译，商务印书馆2003年版。
  - 正确（多作者同国籍）：[德]卡尔·拉伦茨、[德]迪特尔·梅迪库斯：《德国民法总论》，邵建东译，法律出版社2000年版。
  - 正确（多作者不同国籍）：[德]K. 茨威格特、[法]H. 克茨：《比较法总论》，潘汉典等译，法律出版社2003年版。
  - 错误：卡尔·拉伦茨：《法学方法论》，陈爱娥译，商务印书馆2003年版。（外籍作者漏标国籍）

- [ ] **第34条：古籍作者用圆括号（朝代）**
  - 正确：（清）沈家本：《历代刑法考》，邓经元、骈宇骞点校，中华书局1985年版。
  - 正确：（唐）长孙无忌等：《唐律疏议》，刘俊文点校，中华书局1983年版。
  - 正确：（汉）班固：《汉书》，中华书局1962年版。
  - 错误：清·沈家本：《历代刑法考》，中华书局1985年版。（朝代应用圆括号而非间隔号）

- [ ] **第35条：合作作品2-3人全部列明用顿号，多人用"等"**
  - 正确（2人）：张明楷、陈兴良：《刑法学原理》，法律出版社2010年版。
  - 正确（3人）：王利明、杨立新、王轶：《民法学》，法律出版社2017年版。
  - 正确（多人）：梁慧星等：《中国物权法草案建议稿》，社会科学文献出版社2000年版。
  - 错误（多人全部列出）：王利明、杨立新、王轶、程啸、张新宝：《民法学》，法律出版社2017年版。（超过3人应用"等"）
  - 错误（顿号误用逗号）：张明楷，陈兴良：《刑法学原理》，法律出版社2010年版。

- [ ] **第36条：可省略作者的情况（古代典籍、辞书、佚名、个人文集）**
  - 正确（古代典籍）：《唐律疏议》，中华书局1983年版。
  - 正确（辞书）：《辞海》，上海辞书出版社1999年版。
  - 正确（佚名）：《平冤录》，元刻本。
  - 正确（个人文集）：《朱文公文集》，四部丛刊本。
  - 错误（一般现代作品省略作者）：《民法总则》，北京大学出版社2009年版。（除特定情况外一般不可省略作者）

---

## B. 文献名称（第37-40条）

- [ ] **第37条：文献标题用全称加书名号，副标题符号从原文，主标题以问号结尾时用破折号**
  - 正确（一般情况）：王泽鉴：《民法总则》，北京大学出版社2009年版。
  - 正确（副标题）：梁慧星：《民法解释学》，法律出版社2015年版。
  - 正确（主标题以问号结尾）：苏力：《法治及其本土资源》，北京大学出版社2015年版。（主标题本身无问号则正常使用）；如有副标题：王涌：《法律是什么？——二十世纪英美法理学批判阅读》，法律出版社2018年版。
  - 错误：王泽鉴：《民法总则》，北京大学出版，2009年。（应用书名号且用全称）

- [ ] **第39条：图书名称（合订本空格隔开、丛书一卷的省略丛书名、多卷册括号注明、版卷册顺序）**
  - 正确（合订本）：王泽鉴：《民法学说与判例研究》（合订本），北京大学出版社2015年版。
  - 正确（丛书一卷）：王泽鉴：《民法总则》，北京大学出版社2009年版。该书属于"民法研究系列"丛书，引用时不必标注丛书名。
  - 正确（多卷册）：《中国民法典释评》（十卷本），中国人民大学出版社2020年版。
  - 正确（版卷册顺序）：王泽鉴：《民法总则》（第3版），北京大学出版社2012年版。
  - 错误（丛书多卷不标卷次）：《中国民法典释评》，中国人民大学出版社2020年版。（多卷册应注明卷册信息）

- [ ] **第40条：图书版本（初版不标、再版括注"第X版"、外文中译本可括注原书版次）**
  - 正确（初版不标）：王泽鉴：《民法总则》，北京大学出版社2009年版。
  - 正确（再版标注）：王泽鉴：《民法总则》（第3版），北京大学出版社2012年版。
  - 正确（外文中译本标注原书版次）：[德]卡尔·拉伦茨：《法学方法论》（原书第6版），陈爱娥译，商务印书馆2003年版。
  - 错误（初版标注"第1版"）：王泽鉴：《民法总则》（第1版），北京大学出版社2009年版。（初版不标）
  - 错误：王泽鉴：《民法总则》，北京大学出版社2009年第三版。（应用括号括注"第3版"）

## C. 出版信息（第41-47条）

- [ ] **第41条：出版信息（出版社全称不写城市、联合出版用顿号、只写年份不写月份）**
  - 正确：王泽鉴：《民法总则》，北京大学出版社2009年版。
  - 正确（联合出版）：王泽鉴：《民法总则》，北京大学出版社、法律出版社2009年版。
  - 错误（写城市）：王泽鉴：《民法总则》，北京：北京大学出版社2009年版。
  - 错误（写月份）：王泽鉴：《民法总则》，北京大学出版社2009年1月版。（只写年份即可）
  - 错误（出版社简称）：王泽鉴：《民法总则》，北大出版社2009年版。（应用全称）

- [ ] **第42条：文集出版信息（载编者：书名，出版社年份）**
  - 正确：王利明：《物权法定原则》，载梁慧星主编：《民商法论丛》第7卷，法律出版社1997年版，第58页。
  - 正确：苏力：《法治及其本土资源》，载《中外法学》1997年第3期。
  - 错误（缺"载"字）：王利明：《物权法定原则》，梁慧星主编：《民商法论丛》第7卷，法律出版社1997年版。（缺"载"，文集来源标识不清）

- [ ] **第46条：古籍出版信息（选择可靠版本、现代整理本标注出版方式）**
  - 正确（现代整理本）：（清）沈家本：《历代刑法考》，邓经元、骈宇骞点校，中华书局1985年版。
  - 正确（影印本）：（宋）宋慈：《洗冤集录》（影印本），上海古籍出版社1980年版。
  - 错误（古籍引用无版本信息）：《唐律疏议》。（应注明具体版本）

- [ ] **第47条：台湾文献（不用"国立""中央"、公元纪年）**
  - 正确：王泽鉴：《民法总则》，台湾大学法学丛书编辑委员会编辑，2009年版。
  - 正确（年份转换）：王泽鉴：《民法总则》，2009年版。（此处为公元纪年，台湾文献应统一使用公元纪年）
  - 错误（使用"国立"字样）：王泽鉴：《民法总则》，国立台湾大学法学丛书编辑委员会编辑，2009年版。
  - 错误（民国纪年）：王泽鉴：《民法总则》，台湾大学法学丛书编辑委员会编辑，民国98年版。（应转换为公元纪年）

## D. 页码和章节（第48-49条）

- [ ] **第48条：页码格式"第X页"或"第X-X页"，连续页码短横线连接，不连续页码顿号隔开**
  - 正确（单页）：王泽鉴：《民法总则》，北京大学出版社2009年版，第15页。
  - 正确（连续多页）：王泽鉴：《民法总则》，北京大学出版社2009年版，第15-28页。
  - 正确（不连续页码）：王泽鉴：《民法总则》，北京大学出版社2009年版，第15、28、35页。
  - 错误（连续页码用波浪线）：王泽鉴：《民法总则》，北京大学出版社2009年版，第15~28页。（应用短横线）
  - 错误（不连续页码用逗号）：王泽鉴：《民法总则》，北京大学出版社2009年版，第15，28，35页。（应用顿号）
  - 错误（缺"第"字）：王泽鉴：《民法总则》，北京大学出版社2009年版，15页。

- [ ] **第49条：章节（古籍卷册用"卷三十九《定格令》"格式，现代图书用"第五章"格式）**
  - 正确（古籍卷册）：《旧唐书》卷三十九《定格令》。
  - 正确（现代图书）：王泽鉴：《民法总则》，北京大学出版社2009年版，第五章。
  - 正确（古籍卷册加页码）：（清）沈家本：《历代刑法考》，邓经元、骈宇骞点校，中华书局1985年版，卷三。
  - 错误（古籍章节格式混用）：《旧唐书》第39卷《定格令》。（应用"卷三十九"格式）
  - 错误（现代图书不用"第"）：王泽鉴：《民法总则》，北京大学出版社2009年版，五章。

## E. 学位论文（第64条）

- [ ] **格式：作者：《题名》，学位授予单位 年份 学位层级，第X页**
  - 正确（博士论文）：王涌：《法律关系的类型化研究》，中国政法大学2000年博士学位论文，第15页。
  - 正确（硕士论文）：张亮：《我国民法典编纂中的体系问题》，清华大学2015年硕士学位论文，第30页。
  - 错误（漏写学位层级）：王涌：《法律关系的类型化研究》，中国政法大学2000年，第15页。（须标注"博士学位论文"或"硕士学位论文"）
  - 错误（写成"博士论文"而非"博士学位论文"）：王涌：《法律关系的类型化研究》，中国政法大学2000年博士论文，第15页。（应用完整表述"博士学位论文"）

## 附则

> 本清单对应《法学引注手册》第25-49条（中文图书引注）及第64条（学位论文），供检查核对使用。每条规则应结合具体文献类型灵活适用。
