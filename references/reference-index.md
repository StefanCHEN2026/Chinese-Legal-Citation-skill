## 引注语言判定

| 判定依据 | 语言 |
|---------|------|
| 含中文字符（CJK统一表意文字） | 中文 |
| 不含中文，以英文字母开头 | 英文 |
| 不含中文，含假名（ぁ-ん、ァ-ン） | 日文（仅提示） |
| 不含中文，含法文特征词(éd., éditions) | 法文（仅提示） |
| 不含中文，含德文特征词(Aufl., Verlag) | 德文（仅提示） |

## 中文引注类型判定优先级

| 优先级 | 特征 | 类型 |
|-------|------|------|
| 1 | 含 HTTP/HTTPS 链接 | 网络文献 |
| 1 | 含"微信公众号""博客""App" | 网络文献 |
| 2 | "同前注""同上注""同上"开头 | 再次引用 |
| 3 | 含"诉""案"+案号/判决书 | 司法案例 |
| 4 | 含《×法》《×条例》或文号 | 法律文件 |
| 5 | 作者前有（朝代）标记 | 古籍 |
| 6 | 含"博士学位论文""硕士学位论文" | 学位论文 |
| 7 | 含"载《》"+年月日 | 报纸文章 |
| 8 | 含"载《》"+主编/文集 | 文集文章 |
| 9 | 含"载《》"+年份期次 | 期刊文章 |
| 10 | 含"出版社"+"年版"/"第×版" | 图书 |
