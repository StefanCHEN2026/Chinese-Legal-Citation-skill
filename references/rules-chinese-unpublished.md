# 中文未发表文献引注规则

> 对应《法学引注手册》第二版第59-65条

---

## 一、访谈（第60条）

**格式规范**：笔者与×××访谈，时间，地点。

- [ ] 以"笔者与×××访谈"开头
- [ ] 写明访谈时间
- [ ] 写明访谈地点
- [ ] 要素之间用逗号（全角）分隔
- [ ] **需事先征得被访谈人同意**方能引用

> **错误示例**：张三访谈，2023年5月10日。
>
> 错误原因：未注明"笔者与"，缺地点。

> **正确示例**：笔者与王某某访谈，2023年5月10日，北京市海淀区。

---

## 二、私人通讯（第61条）

**格式规范**：×给笔者的电子邮件，时间。

- [ ] 写明通讯人："×××给笔者的电子邮件"
- [ ] 写明时间
- [ ] 必要时可指明通讯方式：书信、电子邮件、即时通讯消息等

> **错误示例**：张三邮件说……
>
> 错误原因：格式不完整，缺"给笔者的"和具体时间。

> **正确示例**：李明给笔者的电子邮件，2023年5月10日。

> **正确示例**（书信）：王五给笔者的书信，2023年3月1日。

---

## 三、内部资料（第62条）

- [ ] **需征得资料所属单位同意**方能引用
- [ ] **涉密资料不得引用**
- [ ] 标注内部资料的基本信息，但应注意保护敏感信息

> **错误示例**：（内部资料，2023年）。
>
> 错误原因：信息过于笼统，无法识别具体资料；且需要确认是否已获授权。

> **正确示例**：《××省法治建设白皮书（内部资料）》，××省司法厅，2023年。
>
> 说明：此处"××"为脱敏处理，实际引用时应标注正式名称。

---

## 四、会议论文（第63条）

**格式规范**：作者：《题名》，会议名称，时间 于地点。

- [ ] 作者后接冒号
- [ ] 题名用书名号
- [ ] 书名号后无标点，直接接逗号
- [ ] 写明会议名称
- [ ] 写明会议时间
- [ ] 写明会议地点，格式为"于××"（"于"前为空格）
- [ ] **已公开发表的**会议论文，原则上引用发表后的版本

> **错误示例**：张某某：《数据产权》，2023年会议论文。
>
> 错误原因：缺会议名称、时间不具体、缺地点。

> **正确示例**：张某某：《数据产权的法律保护》，中国法学会民法学研究会2023年年会，2023年10月 于北京。

> **正确示例**（已发表）：张某某：《数据产权的法律保护》，载《法学研究》2023年第6期，第45页。
>
> 说明：该文已在期刊发表，应引用发表版本而非会议论文版本。

---

## 五、学位论文（第64条）

**格式规范**：作者：《题名》，学位授予单位 年份学位层级论文，第X页。

- [ ] 作者：《题名》格式同一般文章（作者名后全角冒号，题名用书名号）
- [ ] 学位授予单位名称（如"北京大学""清华大学"）
- [ ] 年份加学位层级："××××年博士学位论文"或"××××年硕士学位论文"
- [ ] 页码：第X页
- [ ] **已公开发表的**学位论文，原则上引用发表后的版本（期刊论文或图书）

> **错误示例**：张三：《人工智能法律问题研究》，博士论文，2023年，第12页。
>
> 错误原因：缺学位授予单位，"博士论文"应写为"博士学位论文"；年份应前置。

> **正确示例**：张三：《人工智能法律问题研究》，北京大学 2023年博士学位论文，第12页。

> **错误示例**：李四：《行政裁量权研究》（中国政法大学硕士学位论文），第23页。
>
> 错误原因：学位授予单位与学位层级不应加括号；缺年份。

> **正确示例**：李四：《行政裁量权研究》，中国政法大学 2020年硕士学位论文，第23页。

> **正确示例**（已发表）：李四：《行政裁量权研究》，载《行政法学研究》2021年第3期，第33页。
>
> 说明：学位论文内容已在期刊发表，应引用发表版本。

---

## 六、档案文献（第65条）

**格式规范**：作者：《文献名》（形成时间），保管机构 档案全宗名，档案号。

- [ ] 作者后接冒号（如作者不明可省略）
- [ ] 文献名用书名号
- [ ] 文献形成时间用圆括号（）括注
- [ ] 保管机构名称
- [ ] 档案全宗名
- [ ] 档案号
- [ ] 要素之间用逗号（全角）分隔

> **错误示例**：北京市档案馆，档案号123-456。
>
> 错误原因：缺文献名、缺形成时间、缺档案全宗名。

> **正确示例**：周恩来：《关于法制局工作的意见》（1950年3月5日），北京市档案馆 中共北京市委全宗，档号123-456。

> **正确示例**（无作者）：《华北人民政府关于废除国民党六法全书的指示》（1949年3月），河北省档案馆 华北人民政府全宗，档号GH-01-002。

---

## 综合检查清单

引用未发表文献时，逐项核查以下要素：

- [ ] 访谈类：是否标注"笔者与"、时间、地点
- [ ] 访谈类：是否已征得被访谈人同意
- [ ] 私人通讯：是否写明通讯人、通讯方式、时间
- [ ] 内部资料：是否需要脱敏处理
- [ ] 内部资料：是否已获授权引用
- [ ] 涉密资料：确认不涉及保密内容
- [ ] 会议论文：是否包含了会议名称、时间、地点
- [ ] 会议论文已发表：是否优先引用了发表版本
- [ ] 学位论文：是否写明了学位授予单位
- [ ] 学位论文：年份与学位层级格式是否正确（如"2023年博士学位论文"）
- [ ] 学位论文已发表：是否优先引用了发表版本
- [ ] 档案文献：是否包含文献名、形成时间、保管机构、档案全宗名、档案号
- [ ] 档案文献：形成时间是否用圆括号括注
